# 村田鉄筋株式会社 - 年間工程表 管理システム

## プロジェクト概要
- **プロジェクト名**: 村田鉄筋株式会社 年間工程表管理アプリ (webapp)
- **目的**: 鉄筋工事の現場情報を登録すると、自動で年間工程表(ガントチャート)に反映され、現場ごとの工期・数量・担当者・契約内容・契約金額を一元管理できるWebアプリ
- **対象ユーザー**: 村田鉄筋株式会社の社内担当者(管理者・閲覧者)

## ✅ 実装済みの機能

### 1. 現場登録機能
- 現場名 / 現場担当 / 構造(RC造・S造・SRC造・木造基礎・その他)
- 総数量(kg / t 切替)
- 工期(開始日・終了日)
- 契約区分(支給材 / 材工)ラジオ選択(色分け表示)
- 契約金額(自動カンマ整形・円単位)
- 備考欄(任意)
- **必須項目の未入力チェック** + エラー表示(赤枠 + メッセージ)
- 開始日・終了日の前後関係チェック

### 2. 年間工程表(ガントチャート)
- 1月〜12月を横並びで表示
- 左側: 現場名・担当・構造・総数量・契約区分・契約金額
- 右側: 月単位の工期バー(日付精度で月内位置を計算)
- **支給材は青系 / 材工は緑系**で色分け
- 工期が複数月にまたがる場合は連結バーで表示
- 年をまたぐ工期も対象年内の重なり部分のみ正しく表示
- 四半期ごとに区切り線を強調
- スマホでも横スクロールで全月閲覧可能

### 3. 年間サマリー(工程表上部に表示)
- 年間 総現場数 / 総数量(t) / 総契約金額 / 支給材件数 / 材工件数

### 4. 検索・絞り込み
- 現場名 / 現場担当 / 構造 / 契約区分 / 契約金額レンジ
- 入力即時で工程表が再描画

### 5. 現場一覧 / 編集 / 削除
- 全現場をテーブル表示(現場名・担当で部分一致検索)
- 編集ボタン → 登録フォームへ反映
- 削除ボタン → 確認ダイアログ後に削除
- 契約区分はバッジ色分け表示

### 6. 集計画面
- 全体KPI(年間 総現場数 / 総数量(kg・t) / 総契約金額 / 支給材・材工 件数)
- **担当者別 集計**(現場数・総数量・契約金額合計)
- **月別 集計**(稼働現場数・予定数量・契約金額)…工期日数で按分

### 7. PDF / 印刷出力
- 印刷ボタンで A3横向き 用にレイアウト調整した印刷ビュー
- 印刷時はボタン・入力欄・タブを非表示にし、工程表のみを印刷
- ブラウザの「PDFに保存」で PDF 化可能

### 8. CSV / Excel / JSON 出力
- **CSV**: BOM付きUTF-8(Excelでも文字化けしない)
- **Excel(.xls)**: SpreadsheetML形式・ヘッダー色付き
- 出力項目順: 現場名 / 現場担当 / 構造 / 総数量 / 工期開始 / 工期終了 / 区分 / 契約金額
- **JSONバックアップ / 復元**: 全データのエクスポート・インポート可能

### 9. 権限切替
- ヘッダーで「管理者(編集可)」/「一般ユーザー(閲覧のみ)」を切替
- 一般ユーザーは登録タブ非表示・一覧の編集削除ボタンも非表示

### 10. データ保存
- ブラウザの **localStorage** に保存(タブを閉じても消えない)
- 初回起動時は動作確認用のサンプル現場が3件自動投入される

### 12. デザイン
- 白背景・表形式・大きめ文字・カンマ付き金額
- 月ごとの区切り線・四半期ごとの強調線
- スマホ・タブレット・PC のレスポンシブ対応
- 工程バーは支給材=青 / 材工=緑

## 🌐 機能エントリーURI(画面パス)
すべて単一ページアプリ(SPA)としてタブで切替

| URI / 操作 | 内容 |
|---|---|
| `/` (タブ: 年間工程表) | 年間ガントチャート + サマリー + 絞り込み |
| `/` (タブ: 現場登録) | 新規登録 / 編集フォーム |
| `/` (タブ: 現場一覧) | 現場一覧テーブル(編集・削除) |
| `/` (タブ: 集計) | 全体KPI / 担当者別 / 月別 |
| `/` (タブ: 出力) | PDF・CSV・Excel・JSON 出力 / インポート |
| `/static/style.css` | スタイルシート |
| `/static/app.js` | フロントエンドロジック |
| `/static/favicon.svg` | ファビコン |

## データ構造

```ts
type Site = {
  id: string;            // 一意ID
  name: string;          // 現場名 *
  manager: string;       // 現場担当 *
  structure: string;     // 構造 (RC造/S造/SRC造/木造基礎/その他) *
  quantity: number;      // 総数量(数値) *
  quantityUnit: 'kg'|'t';// 単位
  startDate: string;     // YYYY-MM-DD *
  endDate: string;       // YYYY-MM-DD *
  contractType: '支給材'|'材工'; // 契約区分 *
  amount: number;        // 契約金額(円・数値) *
  memo: string;          // 備考(任意)
  createdAt: string;
  updatedAt: string;
};
```
- 保存先: `localStorage` キー `murata_tekkin_sites_v1`
- 権限: `localStorage` キー `murata_tekkin_role_v1`
- 表示年: `localStorage` キー `murata_tekkin_year_v1`
